what you need to know about malamuter

Malamuter is a Malware scanner for android devices.

Our scanner has multiple features to help guard against malware. Including the following:

Detection Techniques

- Signature Based Detection

- Dynamic Monitoring

- File Extention Block Listing

- Behavior Based Detection

Types of Malware

- rootkits

- Spyware/Keyloggers/Trojan Horses

- Adware

- Worms

- Rootkits

Our goal

We aim to help our users make the best possible decisions regarding the security of their mobile data.

Our scanner utilizes several different open source reposiitories, we align our technical implementation to the MITRE ATT&CK framework.

Other scanners can be very intense on system resources, that's why we put a great deal of focus into ensuring our scanner is optimized as much as possible for mobile devices
